# longxiaclub

AI 导航站项目（Next.js 静态导出）。

## 线上地址

- 主域名: `https://longxiaclub.com`
- Netlify 默认域名: `https://longxiaclub.netlify.app`

## 部署说明

- 部署平台: Netlify
- 构建命令: `npm run build`
- 发布目录: `dist`
- Node 版本: `20`

项目使用 Next.js 静态导出（`output: "export"`），构建产物输出到 `dist` 目录。

## 本地开发

```bash
npm install
npm run dev
```

## 本地构建

```bash
npm run build
```
